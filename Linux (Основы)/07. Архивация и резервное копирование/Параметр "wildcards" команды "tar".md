#flashcards 
***
Используется в [[Команда "tar"]]. Нужен для того, чтобы искать файлы / директории по шаблонам для дальнейшего извлечения из архива ([[Архивирование]]).
Для этого используются [[Групповые символы (wildcards)]], а также [[Частые классы символов]].
***
***Пример***:
>tar xf ../playground2.tar --wildcards 'home/me/playground/dir-* /file-A'
- Извлекаем только файлы, соответствующие указанному пути с групповым символом dir-\*.
